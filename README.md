# LAPD_Crime_Analytics
LAPD crime analysis - Excel, SQL and Power BI dashboard

Description- 
Excel for data overview and brief transformation
MSSQL Server Data Cleaning & Preparation
Power BI Data Modeling
Interactive Dashboard Development

Download the Power BI Dashboard- 
Since .pbix files may exceed GitHub’s upload limit, the dashboard is hosted on Google Drive.

https://drive.google.com/file/d/18ZUMjlLpJ9bEWyCiZ0ubtYwBDLgLPHvq/view?usp=sharing 

Tools & Technologies- 
Excel- Overview
SQL Server – Data cleaning, transformation, and validation
Power BI – Data modeling and visualization
DAX – Custom columns & measures
GitHub – Version control & project hosting
Google Drive – Hosting the PBIX file

How to Use- 
Download the PBIX file from Google Drive
Open in Power BI Desktop
Interact with filters and visuals
Explore crime hotspots, victim demographics, and time patterns

Insights Summary- 
Crime is concentrated heavily in specific areas such as Central, 77th Street, and Pacific.
Highest incidents occur in Street, Single-Family Dwelling, and Multi-Unit Dwelling.
Largest victim Descent falls in Hispanic/Latin/Mexican Descent at 29.49%
Crimes spike during 12:00 with over 35,000 incidents occurring at said time.
Data shows a sharp decline in total incidents from data high in 2022 to 2025
